Dies ist ein Beispiel-Blogbeitrag. Du kannst ihn löschen oder als Vorlage für
neue Beiträge verwenden.

## So schreibst du einen neuen Beitrag

1. Lege im Ordner `_posts/` eine neue Datei an.
2. Der Dateiname **muss** dem Muster `JAHR-MONAT-TAG-titel.md` folgen,
   zum Beispiel `2026-08-03-neue-publikation.md`.
3. Trage oben im sogenannten *Front Matter* (zwischen den drei Bindestrichen)
   Titel und optional Schlagworte ein.
4. Schreibe darunter deinen Text in Markdown.

Sobald du die Datei zu GitHub hochlädst, erscheint der Beitrag automatisch
auf der [Blog-Übersichtsseite]({{ '/blog/' | relative_url }}) und auf der
Startseite unter „Neueste Beiträge" – ganz ohne dass du HTML oder Code
anfassen musst.

> Tipp: Auch von unterwegs kannst du direkt im Browser auf GitHub.com eine
> neue Datei im Ordner `_posts/` anlegen, ganz ohne Programme zu installieren.
